# Система управления складом (Warehouse Management System)

## Описание проекта

Консольное приложение на C# для управления складскими операциями. Система позволяет управлять товарами, создавать заказы, обрабатывать поставки и автоматически уведомлять персонал об изменении уровня запасов.

## Функциональные возможности

- Управление товарами (добавление, просмотр)
- Создание и обработка заказов
- Поставка товаров на склад
- Автоматическое уведомление персонала при достижении порогового значения
- История выполненных заказов
- Логирование всех операций
- Интерактивное консольное меню

## Используемые паттерны

| Паттерн | Назначение |
|---------|------------|
| **Наблюдатель (Observer)** | Автоматическое уведомление сотрудников об изменениях остатков |
| **Одиночка (Singleton)** | Централизованное логирование (класс Logger) |
| **Репозиторий (Repository)** | Отделение логики хранения от бизнес-логики |

## Установка и запуск

### Требования
- .NET SDK 6.0 или выше
- Любая IDE (Visual Studio, Rider, VS Code)

### Команды для запуска

# Клонирование репозитория
git clone https://github.com/your-username/warehouse-system.git

# Переход в директорию проекта
cd warehouse-system

# Сборка проекта
dotnet build

# Запуск приложения
dotnet run
Меню программы
